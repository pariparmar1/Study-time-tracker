# Study-time-tracker
Brief discription: 
This program helps track your study time. In this you can input information and it will display the output of the time needed to study. 

How to Run:
To run this code you will need to input the number of hours you studied. The code will then perform the calculations and tell you how many hours you are on track to study that week. 

Example interaction: 
enter hours studied today: 2
You are on track to study 14 hours this week.
